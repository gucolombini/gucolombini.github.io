Phaser Engine JavaScript Music

Marygold is a bait-and-switch horror browser game developed by me and my girlfriend as one of her University projects. You play as a cute little bunny and must collect the carrots in your farm, but the innocent facade quickly gets dropped when a wolf approaches. While I didn't work on the art nor game design, this time I was responsible for all of the code (and the music!).

I scripted a custom level loading system which let me build the levels using Tiled and allowed me to easily create stages for the game. I was also able to implement a few shaders to allow for some glitchy and distorted visual effects in-game.
